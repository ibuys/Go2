If you’d like to make a backup of your bookmarked hosts, Go2 supports exporting it’s bookmarks as a plain text file, identical to the one it will import above. Go2 will export a plain text file containing one URL per line, ready to be imported again if you need it.

To export a text file, click on “File” and then “Export Bookmarks”.
