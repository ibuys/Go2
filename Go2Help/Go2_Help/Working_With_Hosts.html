Go2 uses Uniform Resource Locators, or URLs, so to add a new bookmark, you simply add the URL into the field labeled URL.

The OK button will be disabled until Go2 determines that a URL that it can work with has been entered. A valid URL has several parts, let’s go over them now.  Here’s an example:

ftp://username:password@servername.com/path


  1. •ftp:// This is the protocol type of the bookmark. This is the portion of the URL that is listed under the “Type” column of the main Go2 window. Go2 can handle any protocol that is registered with OS X, here are a few common ones.

    1. 1.ftp: File Transfer Protocol. One of the oldest protocols, this is handled by Finder by default.

    2. 2.sftp: Secure File Transfer Protocol. You will need a third party application like CyberDuck or Transmit to handle sftp connections.

    3. 3.http: Hypertext Transfer Protocol. Handled by default web browser, probably Safari.

    4. 4.rdp: Remote Desktop Protocol. If you have Windows servers that you need access to, Go2 can store the connection information, and then launch either Microsoft RDC Client, or the excellent open-source CoRD application.

    5. 5.ssh: Secure Shell. Go can manage SSH bookmarks for you, and it’s what I developed it for initially. By default, Go2 will launch Terminal.app to handle ssh connections.

    6. 6.vnc: Virtual Network Computing. Go2 will launch the Screen Sharing application included with OS X to handle VNC connections.

  2. And More… OS X supports lots of URL protocols, either natively or through third party applications. If you know it works, drop it in Go2 for fast access.

  3. •username:password This portion of the URL is optional. The username is used for ftp, ssh, rdp, and vnc connections, and the password portion is optionally used for all except ssh. Currently, Go2 does not encrypt the password before storing it in the database, but that will be addressed in an update in the future.

  4. •servername.com This is the server that you are connecting to. This is also the portion of the URL that is displayed under the “Server Name” column in the main Go window.

  5. •/path This portion of the URL is optional, and used mainly for ftp and sftp connections.
