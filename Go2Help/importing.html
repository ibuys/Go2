Importing Bookmarks

Go2 can import a list of bookmarks saved in a plain text file, one URL per line. For example:

http://www.farmdog.co
sftp://jon@farmdog.co
ssh://jon@farmdog.co

If you copy the URLs above into TextEdit, and save them as a plain text file, you can import them into Go.

To import a text file, click on “File” and then “Import Hosts”.
